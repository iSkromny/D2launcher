Программа для обновления шаблонов, мода sMNS и скачки полной версии мода, а так же запуска всех необходимых программ для игры в PVP на моде sMNS.

Версия 1.4.0.1
- Исправлена ошибка обновления лаунчера.
- Исправлена ошибка отображения списка изменений патча.
- Исправлены ошибки перевода.

Версия 1.4.0.0

**ВАЖНО**! Проект переписан с нуля. С иcпользованием среды **.NET 8 Desktop Runtime**. Если у вас установлена .NET 8 Desktop Runtime, то вы можете скачать версию **light** и запускать у себя как и раньше. 
Если вы запустите **light** версию без среды **.NET 8 Desktop Runtime**, то при первом запуске лаунчера Windows сама предложит вам ее скачать и установить с официального сайта Microsoft.
Если у вас нет .NET 8 Desktop Runtime, то вы можете скачать версию **heavy**. Но(!) версия **heavy** весит 175mb. 

Что нового:


- Удалён чат и торги соответственно.
- Удалена возможность просматривать readme файлы для шаблонов.
- Переработан дизайн.
- Возвращено закрытие программы по клику на кнопку закрытия окна (ранее программа сворачивалась в трей). 
- Добавлен рейтинг Стримерска (получить token можно на сервере Стримерска или в личке у D2-Helper бота).
  - Для получения иконок используются ресурсы игры, поэтому загрузка окон профиля и статистики юнитов занимают некоторое небольшое время.
  - В окне статистики юнитов по умолчанию сортировка стоит по играм. Можно в окне выбрать по победам или winrate.
  - В профиле игрока, на вкладке "Последние матчи" для участников/команд и билдов есть всплывающие подсказки при наведении.
- Добавлена фоновая проверка обновления лаунчера (при доступном обновлении кнопка "Обновить D2launсher" изменит цвет).
- Добавлена фоновая проверка патчей для мода sMNS.
- Добавлен чекбокс "Избранное" которым можно оставить только необходимые вам шаблоны и скрыть остальные.
- Добавлено окно уведомления с информацией о шаблонах (при выходе новых версий сообщает о возможности установить неустановленные, обновить шаблоны из пула избранных, обновить шаблоны из общего пула). Вызывается при старте программы или при нажатии на кнопку "Проверить шаблоны"
- Добавлены темы оформлений.
- Добавлена поддержка языков (переключение происходит без перезапуска).
- В окне "О программе" можно видеть выбранную вами версию мода и последнюю доступную для скачивания версию мода.
- Множество мелких улучшений производительности и исправлений ошибок.



Версия 1.3.0.0 (beta)
Что нового:

- Интегрирован рейтинг с сайта d2mm.ru, таблица лидеров и история игр. На данный момент без авторизации (только ввод вашего ника). В окнах "Таблица D2MM" и "История игр" ники кликабельны и открывают окно профиля игрока.
- Добавлен чат (вход в чат осуществляется через кнопку "Чат"), список пользователей чата онлайн. Необходимо пройти регистрацию. Настоятельная просьба регистрироваться точно с таким же данными, с которыми входить в лобби в игре (для дальнейшего плавного перехода).
- Добавлена возможность торгов (двойной клик по нику игрока в списке). Результат торгов дублируется в отдельный канал чата.
- При старте программы в окне changelog`a выбирается рандомный шаблон.
- В окне "О программе" добавлено наименование актуальной версии и кнопка с последними изменениями патча.
- Закрытие окна программы сворачивает её в трей.
- Исправлены мелкие недочёты.

Версия 1.2.1.0
Что нового:

- Добавлено окно предпросмотра изменений: Перед скачиванием обновления добавлено окно со списком изменений версии.
- Добавлена альтернативная ссылка на полную версию мода: Для тех, у кого сложности со скачиванием с сайта d2mm.ru. Нажатие кнопки "скачать мод" откроет окно с выбором откуда скачать мод.
